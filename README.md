## What I'm working on

### PerfKit

A web app for organizing school performance-assessment information into focused guides, schedules, and study modules.

- Static/PWA-style frontend
- Cloudflare Workers API
- D1 database and R2 file storage
- Admin tools for managing assessments, guides, and upload requests

### FocusForce

A Windows desktop app concept for focus sessions, app/site blocking, break control, and scheduled productivity rules.

- WPF / .NET desktop app
- Allow-list and block-list modes
- Focus sessions, breaks, and recurring schedules
- Practical UX for staying on task without blocking the app itself

### BGEE-Hold2Turbo

A utility for **Baldur's Gate: Enhanced Edition** that helps automate a hold-to-turbo workflow using Cheat Engine speedhack setup.

- Auto-detects the game process
- Opens and attaches the configured Cheat Engine table
- Supports settings for notifications, startup behavior, delays, and cleanup
